  • Note : Pour tous les services la base est http://localhost:8080/Laboratoire10-solution/
Entité Utilisateur Film
Creer un élément PUT /service/utilisateur/creer?nom=Etudiant&motDePasse=Password1&courriel=abc@def.hij PUT /service/film/creer?titre=TITRE&bandeAnnonce=URL_BANDE_ANNONCE&affiche=URL_AFFICHE&resume=RESUME
Afficher un élément GET /service/utilisateur/afficher/ID_UTILISATEUR GET /service/film/afficher/ID_FILM
Modifier un élément POST /service/utilisateur/modifier/ID?nom=NOM&motDePasse=MotDePasse&courriel=abc@def.com POST /service/film/modifier/ID?titre=TITRE&bandeAnnonce=URL_BANDE_ANNONCE&affiche=URL_AFFICHE&resume=RESUME
Effacer un élément DELETE /service/utilisateur/effacer/ID_UTILISATEUR DELETE /service/utilisateur/effacer/ID_FILM
afficher tous les éléments GET /service/utilisateur/afficherListe GET /service/film/afficherListe
